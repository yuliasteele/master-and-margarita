
# Редакция Мастера

Учебный проект по Git и GitHub на основе романа Булгакова.

## Структура репозитория
- /characters — карточки персонажей романа
- /locations — описания мест действия
- /quotes — цитаты из произведения
- /analysis — анализ: темы, символы, связи

## Как пользоваться этим репо

### Навигация по папкам
- **characters/** — карточки персонажей (Воланд, Мастер, Маргарита, Берлиоз и др.)
- **locations/** — описания мест действия (Москва, Иерусалим, Нехорошая квартира)
- **analysis/** — аналитические материалы: темы, символы, связи между персонажами
- **quotes/** — известные цитаты из романа
- **instructions/** — задания и инструкции для участников проекта

### Как найти нужный файл
1. Выбери папку по теме (персонажи, локации, анализ)
2. Открой нужный .md файл — он читается прямо на GitHub
3. Используй поиск по репозиторию (кнопка "Go to file" на главной странице)

### Как внести свой вклад
1. Сделай Fork репозитория
2. Создай ветку: `git checkout -b feature/название`
3. Внеси изменения, сделай commit и push
4. Открой Pull Request и жди ревью
